Ship projects in seconds using prebuilt slices

SwiftStack composes new projects by stitching together compressed, verifiable slices: bases and addons. It verifies SHA-256 hashes, caches artifacts locally, and merges package metadata for consistent outputs.

Browse slices SwiftStack repo

Interactive

Use the TUI wizard or CLI to assemble projects: swiftstack ui, create, build, sync.

Example: swiftstack create --name my-app --base next-base --addons tailwind-ui,auth

How SwiftStack works

  1. Resolve slice aliases from a manifest (bases & addons).
  2. Ensure slice artifacts are cached (download if missing) and verify SHA-256 integrity.
  3. Extract base, apply addons (merge package.json, move files with backup), finalize with npm lockfile update.

Read full details in the SwiftStack README.
